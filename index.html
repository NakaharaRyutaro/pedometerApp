「Resettable_Pedometer（リセット付き歩数計）」アプリ バージョン1.0

　　　　　　　　　　　　　　　　　　　　　　　　　　　作成日　2026年6月18日　
　

apple watch専用の歩数計アプリです。

アップルのヘルスケアアプリの歩数データを利用して、
RESETボタンを押した時から何歩歩いたかを表示します。

日付が変わるとヘルスケアアプリの歩数が０になるので、
本アプリの表示も０に戻ります。

ヘルスケアアプリの歩数は更新周期が長いため、本アプリの
表示周期もヘルスケアアプリと同じ周期になります。

"Resettable _Pedometer" app Version 1.0

Date created　June 18, 2026

This is a pedometer app.
It's exclusively for Apple Watch.

It uses step data from Apple's Health app to display the number of
steps taken since the RESET button was pressed.

When the date changes, the step count in the Health app resets to 0,
so this app's display also resets to 0.

Because the Health app's step count has a long update cycle, this
app's display cycle is the same as the Health app's.
